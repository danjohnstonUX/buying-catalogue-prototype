Order C000001-01

Complete the following steps to create an order summary. Your progress will be saved as you complete each section so you can leave and come back to your order later.

1. Prepare order

  • Order description {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Provide a description of your order.
  • Call-off Ordering Party contact details {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Provide information about the primary contact for
    your order.
  • Supplier information and contact details {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Find the supplier you want to order from and
    select a supplier contact.
  • Timescales for Call-off Agreement {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Provide the approximate start date, the
    length of the contract and its initial period.

2. Add solution and services

  • Solutions and services {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Select a solution or services, Service Recipients,
    prices and quantities.
  • Funding source {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Review how you’ll be paying for your order.

3. Complete contract

  • Security measures {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Select if you require any special security measures.
  • Implementation plan and milestones {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Select if you’re using the standard implementation
    plan or have agreed a bespoke one.
  • Associated Service billing and requirements {{ tag({ text: "Not started", classes: "nhsuk-tag--grey" })}}
    Select how you’d like to be billed for any Associated
    Services in your order.
  • Data processing information {{ tag({ text: "Not started", classes: "nhsuk-tag--grey" })}}
    Select if you want to make any changes to the
    default data processing information.

4. Finish order

  • Review and confirm order {{ tag({ text: "Not started", classes: "nhsuk-tag--grey" })}}
    Check the information you’ve provided is correct
    and complete your order summary.
