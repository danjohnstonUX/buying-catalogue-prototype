Overview Implementation Plan

Where a number of Service Instances will be undergoing implementation planning and/or actual implementation the Call Off Ordering Party may require an Overview Implementation Plan which will set out, as a minimum, the following for each of the Service Instances which have yet to Achieve Milestone M2:

  • the Milestone Dates for Milestones M1 and M2;
  • the start and end dates for any activity associated with the migration of data from the solutions which the Catalogue Solution is replacing to the Catalogue Solution;
  • the start and end dates for the training activity.
