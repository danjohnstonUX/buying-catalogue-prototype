Order C000001-01 - NHS Hull CCG

Complete the following steps to create an order summary. Your progress will be saved as you complete each section so you can leave and come back to your order later.

Description

This description text will only appear when the 'order description' section has been completed in Step 1.

1. Prepare order

  • Order description {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Provide a description of your order.
  • Call-off Ordering Party contact details {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Provide information about the primary contact for
    your order.
  • Supplier information and contact details {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Find the supplier you want to order from and select
    a supplier contact.
  • Timescales for Call-off Agreement {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Provide the approximate start date, the length of
    the contract and its initial period.

2. Add solution and services

  • Solutions and services {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Select a solution or services, Service Recipients, prices
    and quantities.
  • Planned delivery dates {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Enter the planned delivery dates for what you’re
    ordering.
  • Funding source {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Review how you’ll be paying for your order.

3. Supporting documents

  • Implementation plan milestones {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Download the implementation plan milestones
    template for the supplier to complete.
  • Associated Service billing and requirements {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Select how you’d like to be billed for any Associated
    Services in your order.
  • Data processing information {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Download the data processing information
    template for the supplier to complete.

4. Finish order

  • Review and confirm order {{ tag({ text: "Completed", classes: "nhsuk-tag--green" })}}
    Check the information you’ve provided is correct
    and complete your order summary.
