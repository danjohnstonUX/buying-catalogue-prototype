Chemixal Web, 2.0

Solution description

Summary

Chemixal Web is one of the most popular digital clinical systems in the UK. Our system is designed to be a complete solution that focuses on usability and clinical safety. This Solution supports joined-up working accross all care settings, helping you work towards the target of dissolving boundaries.

About Solution

Chemixal Web has been used across the UK since 1980. With nearly 300,000 patient using it annually, it is one of the most relied upon Solutions in the market.

It is an intelligent CMS platform designed to support the secure management and classification of all forms unstructured electronic documents including those created by scanning paper documents. It also enables processing of documents and matching documents with Patients.

By Documenting expertise in document imageing, high-speed mail automationg and material handling with our leading electronic and records management software solution, healthcare organisations will benefit from a holistic document management service tailored to their requirements.

www.chemixal-pro.co.uk/chemixal-web-2

Features

  • View and add to clinical records, including shared data
  • View daily appointments and diary entries
  • View a map of patient appointment locations
  • Update clinical records via templates
  • Add user tasks
  • Book follow-up appointments
  • Create and amend appointment schedules
  • Add new medication
  • View and enter data on and offline
  • View document attachments

    Epic 1 - HR Manager

    As a GP Practice Manager
    I want to access the HR Manager tool
    So that I can utilise the tool to analyse employee activity

    Acceptance criterion 1
    Given that a Practice Manager has the correct administration rights
    When the HR Manager tool is accessed
    Then I can access the HR Manager tool
    And I can create reports

    Acceptance criterion 2
    Given that an employee has the correct administration rights
    When the HR Manager tool is accessed
    Then I can access the HR Manager tool
    And I can access reports

Hybrid

Summary

Our solution uses a combination of private and public cloud suppliers. We store all of our patient confidential data in a data centre that we own and manage. Data center hosting model

Our managed data centre is hosted in two separate geographical locations, they both comply to the highest standards to ensure that even if one of our data centres suffers an outage, we can ensure that your daily activities are not interrupted. We also create a back up of all of our data every evening and store it separately, so in the result of any catastrophic failure, we can ensure that patient confidential information is kept secure.
