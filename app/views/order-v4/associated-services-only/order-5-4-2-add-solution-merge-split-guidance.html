Guidance for {{data['cat_solution']}}

A Merger is where two (or more) practices join into a single practice with a combined patient list and single patient database. The Lead Practice in the Merger will retain it's existing ODS Code and the merged practice(s) will not retain it's previous ODS code, but may have a new Branch ODS Code(s) assigned if it remains as a separate location/premises.

For example the Lead Practice will retain it’s ODS code A99999 and the “branch site” will become A99999001.

Branch sites are just treated as part of the main site for charging (not needing a separate call-off agreement) so any orders that deal with branches will just reference the main site.

A practice closure may occur where a practice shuts down (e.g. due to retirement, or financial insolvency, etc.) the transfer of patients is classed as a merger
