This Catalogue Solution has the following features:

  • Realtime admin appointments booking (incl. COVID-19 vaccinations) at PCN/CCG/practice level
  • Patient facing web access (and apps)
  • Secure video consultation interface
  • Secure instant messaging
  • Secure image upload capability
  • Clinician safety tools in consultation
  • Intelligent alerts and auto-templates to capture outstanding QOF information
  • Live Chat functionality on apps and web with 8am-8pm support for all users
  • Organisation level roles and permissions, built in partnership with an STP
